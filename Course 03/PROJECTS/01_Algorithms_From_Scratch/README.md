# Project 01: Implement ML Algorithms from Scratch | المشروع 01: تنفيذ خوارزميات تعلم الآلة من الصفر

## Overview | نظرة عامة

Implement core machine learning algorithms from scratch using only NumPy, demonstrating understanding of the mathematical foundations.

**Learning Objectives:**
- Implement linear algebra operations
- Build regression algorithms
- Create classification algorithms
- Understand gradient descent
- Apply mathematical concepts

---

## Requirements | المتطلبات

### Functional Requirements
1. **Linear Regression**
   - Implement from scratch using NumPy
   - Use gradient descent for optimization
   - Calculate R² and other metrics

2. **Logistic Regression**
   - Implement from scratch
   - Use gradient descent
   - Apply sigmoid function

3. **PCA Implementation**
   - Implement eigenvalue decomposition
   - Calculate principal components
   - Apply dimensionality reduction

4. **Testing and Validation**
   - Test on real datasets
   - Compare with scikit-learn
   - Validate correctness
