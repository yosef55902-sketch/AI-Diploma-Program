# Project 02: PCA Implementation and Visualization | المشروع 02: تنفيذ PCA والتصور

## Overview | نظرة عامة

Implement Principal Component Analysis from scratch and create visualizations of the process.

**Learning Objectives:**
- Implement PCA from scratch using NumPy
- Understand eigenvalue decomposition
- Visualize PCA transformation
- Apply dimensionality reduction
- Compare with scikit-learn implementation

---

## Requirements | المتطلبات

### Functional Requirements
1. **PCA Implementation**
   - Implement from scratch using NumPy
   - Calculate covariance matrix
   - Perform eigenvalue decomposition
   - Extract principal components
   - Transform data

2. **Visualization**
   - Visualize original data
   - Show principal components
   - Display variance explained
   - Create transformation animations
   - Compare before/after

3. **Analysis**
   - Calculate explained variance
   - Determine optimal components
   - Compare with scikit-learn
   - Analyze reconstruction error
