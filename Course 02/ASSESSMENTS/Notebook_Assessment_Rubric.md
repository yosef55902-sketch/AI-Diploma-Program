# Notebook Assessment Rubric | معايير تقييم الدفاتر

## General Assessment Criteria | معايير التقييم العامة

**Total Points: 100 per notebook**

---

## 1. Understanding (40 points) | الفهم (40 نقطة)

### Excellent (36-40 points)
- ✅ Demonstrates deep understanding of concepts
- ✅ Can explain concepts in own words
- ✅ Connects concepts to real-world applications
- ✅ Shows understanding of relationships between topics

### Good (30-35 points)
- ✅ Good understanding of main concepts
- ✅ Can explain most concepts
- ✅ Some connection to applications

### Satisfactory (24-29 points)
- ✅ Basic understanding
- ✅ Can explain some concepts
- ✅ Limited connection to applications

### Needs Improvement (0-23 points)
- ❌ Poor understanding
- ❌ Cannot explain concepts
- ❌ No connection to applications

---

## 2. Code Implementation (30 points) | تنفيذ الكود (30 نقطة)

### Excellent (27-30 points)
- ✅ All code examples completed correctly
- ✅ Code runs without errors
- ✅ Code is clean and well-structured
- ✅ Experiments with variations

### Good (24-26 points)
- ✅ Most code completed correctly
- ✅ Minor errors that are fixed
- ✅ Code mostly clean

### Satisfactory (18-23 points)
- ✅ Some code completed
- ✅ Some errors present
- ✅ Code works but could be better

### Needs Improvement (0-17 points)
- ❌ Many code examples incomplete
- ❌ Multiple errors
- ❌ Code doesn't run

---

## 3. Exercises & Practice (20 points) | التمارين والممارسة (20 نقطة)

### Excellent (18-20 points)
- ✅ All exercises completed
- ✅ Attempts bonus/challenge problems
- ✅ Shows creative solutions
- ✅ Experiments beyond requirements

### Good (15-17 points)
- ✅ Most exercises completed
- ✅ Good effort on exercises
- ✅ Some experimentation

### Satisfactory (12-14 points)
- ✅ Basic exercises completed
- ✅ Limited experimentation

### Needs Improvement (0-11 points)
- ❌ Many exercises incomplete
- ❌ No experimentation

---

## 4. Participation & Engagement (10 points) | المشاركة والتفاعل (10 نقاط)

### Excellent (9-10 points)
- ✅ Actively engaged in learning
- ✅ Asks thoughtful questions
- ✅ Helps other students
- ✅ Goes beyond requirements

### Good (7-8 points)
- ✅ Engaged in learning
- ✅ Completes work on time
- ✅ Some participation

### Satisfactory (5-6 points)
- ✅ Basic engagement
- ✅ Completes minimum requirements

### Needs Improvement (0-4 points)
- ❌ Low engagement
- ❌ Missing work
- ❌ No participation

## Assessment Methods | طرق التقييم

1. **Quizzes** (30% of notebook grade)
   - See QUIZZES folder
   - Test understanding of concepts

2. **Code Exercises** (40% of notebook grade)
   - Completion of notebook exercises
   - Code quality and correctness

3. **Practice Problems** (20% of notebook grade)
   - Additional practice exercises
   - Creative applications

4. **Participation** (10% of notebook grade)
   - Class participation
   - Questions and engagement
