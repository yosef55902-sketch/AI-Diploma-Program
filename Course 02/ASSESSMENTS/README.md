## Tips for Instructors | نصائح للمدرسين

1. **Quizzes:**
   - Administer after notebook completion
   - Allow 30-50 minutes
   - Review answer keys beforehand
   - Adjust difficulty if needed

2. **Projects:**
   - Use rubric consistently
   - Provide feedback using template
   - Encourage creativity in bonus features

3. **Notebooks:**
   - Assess regularly (not just at end)
   - Provide timely feedback
   - Use participation component to encourage engagement
