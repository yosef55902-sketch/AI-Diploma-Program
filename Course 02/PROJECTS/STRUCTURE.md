## What's in Each Project Folder? | ما في كل مجلد مشروع؟

### For Each Project:

1. **README.md** - Complete Project Description
   - Overview and learning objectives
   - Requirements (functional & technical)
   - Deliverables
   - Evaluation criteria
   - Resources
   - Submission guidelines

2. **Implementation_Guide.md** - Step-by-Step Guide
   - Detailed implementation steps
   - Code structure recommendations
   - Testing guidelines
   - Troubleshooting tips
   - Code examples

3. **Template/** - Starter Code
   - Code templates with TODO comments
   - Function signatures ready to fill
   - Example structure
   - README explaining how to use

## How to Use | كيفية الاستخدام

### For Students:

1. **Choose a Project:**
   - Read `PROJECTS/README.md` for overview
   - Go to project folder (e.g., `01_Pathfinding_Game/`)
   - Read project `README.md` for details

2. **Get Started:**
   - Read `Implementation_Guide.md` for steps
   - Copy template from `Template/` folder
   - Follow the guide step-by-step

3. **Work on Project:**
   - Use template as starting point
   - Follow implementation guide
   - Test incrementally

### For Instructors:

1. **Assign Projects:**
   - Direct students to `PROJECTS/README.md`
   - Students choose from numbered folders
   - Each folder has complete materials

2. **Provide Support:**
   - Reference implementation guides
   - Use templates for examples
   - Check project READMEs for requirements
