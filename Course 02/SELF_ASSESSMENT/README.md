# Self-Assessment Materials | مواد التقييم الذاتي

This folder contains self-assessment checkpoints to help students track their progress.

---

## Available Checkpoints | نقاط التقييم المتاحة

### Checkpoint 00: After Notebook 00
**File:** `Checkpoint_After_Notebook_00.md`

Complete after finishing Python Libraries notebook.

**Purpose:**
- Verify understanding of libraries
- Check readiness for Notebook 01
- Identify areas to review

---

### Checkpoint 05: After Notebook 05
**File:** `Checkpoint_After_Notebook_05.md`

Complete after finishing all notebooks (final checkpoint).

**Purpose:**
- Overall course knowledge check
- Project readiness assessment
- Final self-evaluation

---

## How to Use | كيفية الاستخدام

### For Students:

1. **After completing each notebook:**
   - Complete relevant checkpoint
   - Be honest about your understanding
   - Review areas where you're weak
   - Don't move forward until you're ready

2. **Before starting projects:**
   - Complete final checkpoint (Checkpoint 05)
   - Assess your readiness
   - Review weak areas
   - Plan your project approach

3. **Throughout the course:**
   - Use checkpoints to track progress
   - Identify learning gaps early
   - Get help when needed

## Tips | نصائح

1. **Be Honest:**
   - Don't check boxes you're not sure about
   - Honest assessment helps you learn better

2. **Review Regularly:**
   - Don't wait until the end
   - Use checkpoints after each notebook

3. **Take Action:**
   - If you identify weak areas, review them
   - Don't just check boxes and move on
   - Practice until you're confident

4. **Ask for Help:**
   - If you're struggling, ask your instructor
   - Use checkpoints to identify what to ask about
